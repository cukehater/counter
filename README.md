# 🐧 Feathers McGraw Counter

---

![데모이미지](./counter.png)

페더 맥그로우 캐릭터의 숫자 카운터입니다. <br />
특정 개수 달성 시 연출되는 빵빠레 효과도 있으니 한번 확인해보세요 🎉

1. 저장소 복제

```
$ git clone https://github.com/cukehater/counter.git
```

2. 디펜던시 설치

```
$ npm install
```

3. 실행

```
$ npm run dev
```
